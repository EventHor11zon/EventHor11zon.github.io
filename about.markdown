- I go by various nicknames, most of which are *EventHor1zon*, *EventHor11zon*, *Event_Hor1zon*... You get the gist. The name I picked isn't quite unique, unfortunately.
- I'm a first year electrical engineering student from Türkiye.
- I have been making my own video games ever since 2018. For a long time, I did it in websites with visual coding such as Scratch and TurboWarp[^1], but eventually learned Python, Lua and a bit of the C languages. Currently, I use Godot Engine for my projects.

### Stuff I'm Familiar With
- Python (Pygame for making games)
- Lua (Love2D for making games)
- Godot Game Engine (GDScript)
- MATLAB
- Linux (overall use of it)
- Scratch & Turbowarp

That’s pretty much everything I can tell about myself! I publish my completed (or at least playable) games in [Itch.io](https://event-hor11zon.itch.io/), so I recommend you check my profile there!

[^1]: The two may seem like the same thing, but TurboWarp is a modified version of Scratch 3.0 that gets rid of the many limitations of it (such as the screen size and framerate limitations) and introduces many more features to it. People make amazing 3D projects with it, even game engines!
